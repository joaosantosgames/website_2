Estilizando Bordas

Aqui como podemos observar, temos 4 tipos de bordas sendo as:

  1. Solid: padrão e muito usado< ela é uma borda lisa(linha: amarelo)
  2. Dashed: Ela é uma borda listrada quadriculado usado para diferenciar topicos de uma tabela(linha: azul)
  3. Dotted: Ela é uma borda listrada circular tambem usado para diferenciar topicos de uma tabela(linha: verde)
  4. Double: É uma borda com duas linhas usado como margem(linha: roxo)

Margins & Paddings (Espaçamentos Externos e Internos)

Margins

São espaçamentos externos que vai fazer uma distancia de um elemento para outro no mesmo nivel ou de um outro nivel.

Estamos separando as margins com valores diferentes no mesmo comando

Exemplo: margin: 10px 20px 30px 40px;

Aqui esta sendo usado a margin X e Y e/ou eles em diferentes comandos

Exemplo: margin: 20px 30px;

Estamos usando a margin com todos os lados iguais aqui

Exemplo: margin: 30px;

Paddings

São espaçamentos internos usado para ajustar texto ou afastar ele na borda

Aqui esta sendo usado o padding em todos os lados iguais

Exemplo: padding: 50px;

Aqui estamos usando o padding nas posições X e Y(X: comprimento; Y: largura;)

Exemplo: padding: 50px 100px;

Aqui o padding esta com medidas diferentes, porem, dentro do mesmo comando

Exemplo: padding: 10px 20px 30px 40px;

Aqui o padding esta com medidas diferentes, porem, em comandos separados

Exemplos: padding-top: 10px; padding-bottom: 20px;
